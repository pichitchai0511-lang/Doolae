# Doolae Project

Doolae is a comprehensive health tracking application leveraging SwiftUI and CoreData to manage user health entries and provide insights into health risks.

## Features
- Daily logging of health entries
- Visualization of health risks
- CoreData based persistent storage
- User-friendly social feed

## Getting Started
- Clone the repository
- Open `Doolae.xcodeproj` in Xcode
- Build and run the application

## Requirements
- iOS 15.0+
- Xcode 13+

## Installation
Run the following command in terminal:
```
swift package resolve
```
